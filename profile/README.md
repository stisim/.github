## Hi there 👋

Welcome to STIsim!
STIsim is an agent-based framework for modeling the spread of sexually-transmitted diseases among agents via dynamic transmission networks.

It's built on the Starsim modeling platform, but has specific customizations for modeling STIs, including detailed sexual networks, disease models, and interventions.

See stisim.org for more information, or go straight to the docs if you're ready to dive in.

We are strong believers in open source code, and we always welcome contributions from the community. Please feel free to explore, add code, and ask questions!
